Description du fichier.

Ce fichier contient les implémentations des fonctions nécessaires pour manipuler et effectuer des opérations sur les structures de temps timespec.

Ce fichier fournit des fonctions pour convertir une structure timespec en microsecondes, additionner et soustraire deux structures timespec.
