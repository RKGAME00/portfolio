STM32 - Microcontrôleurs ARM

Famille de microcontrôleurs 32-bits haute performance de STMicroelectronics

Microcontrôleur ARM Cortex 32-bit

STM32 - Architecture ARM Cortex

Les microcontrôleurs STM32 de STMicroelectronics offrent une large gamme de solutions 32-bits basées sur l'architecture ARM Cortex. Ils combinent performance élevée, faible consommation et rich périphériques pour applications embarquées avancées.

STM32 Discovery

Familles STM32 Utilisées

STM32F4 Discovery

Performance

Carte de développement basée sur STM32F407VGT6, idéale pour applications haute performance avec FPU.

Core

ARM Cortex-M4F 168MHz

Mémoire

1MB Flash, 192KB RAM

FPU

Virgule flottante matérielle

Périphériques

USB, Ethernet, CAN, DAC

STM32L4 Nucleo

Ultra Low Power

Microcontrôleur ultra-basse consommation pour applications IoT et sur batterie, avec DSP intégré.

Consommation

100 nA en Standby

Connectivité

USB, CAN, LPUART

DSP

Instructions DSP intégrées

Capteurs

Température, RTC, ADC 16-bit

STM32H7 Nucleo

High Performance

Série haute performance avec double cœur Cortex-M7/M4, pour applications intensives et IA embarquée.

Performance

480 MHz Cortex-M7

Dual Core

M7 + M4 asymétrique

Mémoire

2MB Flash, 1MB RAM

Graphics

Chroma-ART, JPEG, LTDC
